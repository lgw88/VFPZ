How to: Change Page Settings for Reports
See Also 

You can change the default page settings for a report or label. For example, you can change the left margin, the default font for the page layout, the paper size and source, and the page orientation.

Note

When you change the paper size and page orientation, make sure the page orientation matches the paper size. For example, if you select an envelope, you should set the page orientation to Landscape.

Note

When you set the default font for the page layout, the setting applies only to new report controls that you add to the report. For more information, see How to: Change Font Settings in Report Controls.

To set the left margin for the page

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, click Properties.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder, the Report Page Setup dialog box opens instead. If _REPORTBUILDER is set to a third-party builder, a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Report Page Setup Dialog Box.

  3. In the Report Properties dialog box, click the Page Layout tab if it is not selected.

  4. In Left margin box, type or select a value for the left margin.

  5. Click OK.

    Note

    When you change the left margin for a page with columns, the width for the column changes automatically to accommodate the new margin.

For more information, see Page Layout Tab, Report Properties Dialog Box (Report Builder).

To change the default font for the page layout

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, click Default Font.

  3. In the Font dialog box, select the font style and size you want, and then click OK.

For more information, see Font Dialog Box.

You can also change the default font from the Report menu by clicking Properties to open the Report Properties dialog box. Click the Page Layout tab, and in the Default font area, click the ellipsis (…) button to open the Font dialog box.

Note

When you use this method to adjust the default font for a report, the Report Builder notes the Script you selected for the font in the dialog box and adds this information to the report layout. You see the Use font script checkbox checked when you return to the Page Layout tab of the Report Properties dialog, and the label showing your selected font, font size, and style, also shows the numeric value representing this script. If you don't want to save this information, uncheck the checkbox. For more information, see Page Layout Tab, Report Properties Dialog Box (Report Builder).

Note

If the _REPORTBUILDER system variable is not set to the default Report Builder or is set to a third-party builder, the Report Page Setup dialog box displays or a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Report Page Setup Dialog Box.

To change the paper size or source

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, click Properties.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder, the Report Page Setup dialog box opens instead. If _REPORTBUILDER is set to a third-party builder, a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Report Page Setup Dialog Box.

  3. In the Report Properties dialog box, click the Page Layout tab.

  4. On the Page Layout tab, click Page setup.

    The Page Setup dialog box opens.

  5. In the Size box, select the paper size you want.

  6. In the Source box, select the paper source you want.

  7. When you are finished, click OK.

For more information, see Page Layout Tab, Report Properties Dialog Box (Report Builder).

To change the page orientation

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, click Properties.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder, the Report Page Setup dialog box opens instead. If _REPORTBUILDER is set to a third-party builder, a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Report Page Setup Dialog Box.

  3. In the Report Properties dialog box, click the Page Layout tab.

  4. Click Page setup.

    The Page Setup dialog box opens.

  5. In the Orientation area, click Portrait or Landscape, and then OK.

For more information, see Page Layout Tab, Report Properties Dialog Box (Report Builder).
