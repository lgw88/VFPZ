Visual FoxPro 9.0 SP2
Wizard Selection Dialog Box
See Also

Displays a list of wizards from which you can choose. This dialog box is displayed when you choose Wizards from the Tools menu and then choose Query, Form, Report, or All from the submenu.

Select the wizard you would like to use

Displays all Visual FoxPro wizards available for a particular item. For instance, Visual FoxPro has three different report wizards.

Description

Gives a brief description of the currently selected wizard.
