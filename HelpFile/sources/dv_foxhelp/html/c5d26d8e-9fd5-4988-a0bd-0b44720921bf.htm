Visual FoxPro 9.0 SP2
SET INTENSITY Command
See Also 
 

 

Included for backward compatibility. Use SET COLOR OF SCHEME instead.

This command is undocumented in the original VFP9 help file,
but the following documentation was found in the FoxPro for Windows 2.6 help, and may still be valid:

Determines whether or not FoxPro uses the enhanced screen color attribute for the display of editing fields.

SET INTENSITY ON | OFF

Remarks


SET INTENSITY is included for backward compatibility. Use SET COLOR OF SCHEME instead.

In FoxPro for MS-DOS, SET INTENSITY specifies whether the enhanced screen attribute is used for fields displayed with @ ... GET, APPEND, BROWSE, CHANGE or EDIT.

In FoxPro for Windows, SET INTENSITY specifies if a Windows color setting is used for editing regions created with @ ... GET.

In FoxPro for Macintosh, SET INTENSITY is ignored.

ON
 In FoxPro for MS-DOS, the enhanced screen attribute is used to highlight fields if SET INTENSITY is ON.

 In FoxPro for Windows and FoxPro for Macintosh, the background color of an editing region created with @ ... GET is determined by the Windows color setting for button faces when INTENSITY is ON. This color can be changed by choosing the Colors icon in the Windows Control Panel.

 ON is the default setting.

OFF
 The normal screen attribute is used if SET INTENSITY is OFF.
