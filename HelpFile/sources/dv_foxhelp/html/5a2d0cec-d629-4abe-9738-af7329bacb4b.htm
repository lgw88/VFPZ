Visual FoxPro 9.0 SP2
SET DISPLAY Command
See Also 
 

 

Included for backward compatibility. Use the font properties of the _SCREEN System Variable instead.

This command is undocumented in the original VFP9 help file,
but the following documentation was found in the FoxPro for Windows 2.6 help, and may still be valid:

Enables you to change the current display mode on monitors that support different modes.

SET DISPLAY TO CGA | COLOR | EGA25 
	| EGA43 | MONO | VGA25 | VGA50

Remarks


In FoxPro for MS-DOS, SET DISPLAY enables you to switch between different display modes on monitors that support different modes.

In FoxPro for Windows and FoxPro for Macintosh, SET DISPLAY changes the font size for the main FoxPro window. The size of the main FoxPro window is increased if necessary to accommodate the number of lines for the option you specify. If the graphical status bar appears when you issue SET DISPLAY, it is turned off.

The error message "Display mode not available" appears if an option isn't supported by your video hardware.

The SET MESSAGE line is reset to the last line of the main FoxPro window whenever SET DISPLAY is issued.

CGA
 In FoxPro for MS-DOS, switches display mode to CGA.

 In FoxPro for Windows and FoxPro for Macintosh, switches the font size for the main FoxPro window to 9-point.

COLOR
 In FoxPro for MS-DOS, switches display mode to color.

EGA25
 In FoxPro for MS-DOS, switches display mode to EGA and 25 lines.

 In FoxPro for Windows and FoxPro for Macintosh, switches the font size for the main FoxPro window to 9-point and the main FoxPro window size to 25 lines.

EGA43
 In FoxPro for MS-DOS, switches display mode to EGA and 43 lines.

 In FoxPro for Windows, switches the font size for the main FoxPro window to 7-point and the main FoxPro window size to 50 lines.

 In FoxPro for Macintosh, switches the font size for the main FoxPro window to 8 point.

MONO
 In FoxPro for MS-DOS, switches display mode to monochrome.

VGA25
 In FoxPro for MS-DOS, switches display mode to VGA and 25 lines.

 In FoxPro for Windows and FoxPro for Macintosh, switches the font size for the main FoxPro window to 9-point and the main FoxPro window size to 25 lines.

VGA50
 In FoxPro for MS-DOS, switches display mode to VGA and 50 lines.

 In FoxPro for Windows, switches the font size for the main FoxPro window to 7-point and the main FoxPro window size to 50 lines.

 In FoxPro for Macintosh, switches the font size for the main FoxPro window to 8 point.
