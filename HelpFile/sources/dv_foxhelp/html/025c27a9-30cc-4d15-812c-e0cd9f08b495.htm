How to: Delete Bands from Reports
See Also 

You can delete the Title, Summary, and Detail bands from your report or label. For information about deleting data groups, see How to: Delete Data Groups in Reports.

Note

You cannot reduce the number of Detail bands to zero. The page layout must always contain one Detail band. When you reduce the number of Detail bands, the highest-numbered band, which is the band added last and appears at the lowest level of the layout, is deleted.

Caution

If a Detail band contains items, the designer prompts you to confirm deleting the band. However, the designer does not prompt you for confirmation when clearing the Title band or Summary band check box, which deletes the band and any controls it contains.

To delete a Title, Summary, or Detail band

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, click Optional Bands.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder or is set to a third-party builder, the Optional Bands dialog box displays or a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Optional Bands Dialog Box.

  3. In the Report Properties dialog box, click the Optional Bands tab if it is not selected.

  4. On the Optional Bands tab, select the bands you want to remove:

    • To delete the Title band, clear the Report has title band check box.

    • To delete the Summary band, clear the Report has summary band check box.

    • To delete Detail bands, in the Detail bands list, select the Detail band you want, and click Remove. When the designer prompts you to confirm deletion, click Yes to confirm deletion.

  5. When you are finished, click OK.

For more information, see Optional Bands Tab, Report Properties Dialog Box (Report Builder).
