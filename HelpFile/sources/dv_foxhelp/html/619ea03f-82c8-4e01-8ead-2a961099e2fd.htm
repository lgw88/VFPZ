Visual FoxPro 9.0 SP2
Cannot build a DLL without OLE public classes (Error 2002)

The BUILD DLL command requires that OLE public classes be defined.

  • You chose the wrong option from the Build Options dialog box.

    Choose another action, such as Build Application.

  • No OLE public classes are defined.

    Define one or more OLE public classes. For more information, see the Class Info dialog box or the OLEPUBLIC clause of the DEFINE CLASS command.
