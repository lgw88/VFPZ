Reports Tab, Application Builder
See Also

On this tab of the Application builder, you can specify friendly names for reports in your application.

Name

Makes it possible to specify the friendly name for the selected report.

Appear in Print Reports dialog

Specifies whether the friendly name of the selected report appears in the Print Reports dialog box of your application.

Add

Makes it possible to add an existing report to your application.

Edit

Makes it possible to edit the selected report in the Report designer. For more information, see Report Designer.

Remove

Removes the selected report from your application.

If you choose OK, the builder closes, applying the property settings from all tabs.
