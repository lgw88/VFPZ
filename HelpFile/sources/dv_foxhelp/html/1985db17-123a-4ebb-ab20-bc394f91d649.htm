Visual FoxPro 9.0 SP2
Cannot find view "name" in the current database (Error 1563)

The name you have specified is not the name of a view in the current database. Check that the name entered is correct and that the current database is the required database. Use DISPLAY VIEWS or LIST VIEWS to confirm the names of the existing database views.

If the view name is in the list, the .dbc index might be corrupted.

To rebuild the index

  1. Close the database.

  2. Delete the index file (database_name.DCX).

  3. Re-open the database.

  4. Re-create the index.
