Visual FoxPro 9.0 SP2
DISPLAY DLLS Command
See Also 
 

Displays information about shared library functions registered in Visual FoxPro with DECLARE – DLL.

DISPLAY DLLS [TO PRINTER [PROMPT] | TO FILE FileName [ADDITIVE]] [NOCONSOLE]

Parameters

TO PRINTER [PROMPT]

Directs output from DISPLAY DLLS to a printer.

In Visual FoxPro, you can include the optional PROMPT clause to display a Print dialog box before printing starts. Place PROMPT immediately after TO PRINTER.

TO FILE FileName

Directs output from DISPLAY DLLS to the file specified with FileName. If the file already exists and SET SAFETY is ON, Visual FoxPro displays a prompt asking if you want to overwrite the file.

ADDITIVE

Appends to the end of the named file. If you omit ADDITIVE, then the file is overwritten with the value of the expression.

NOCONSOLE

Suppresses output to the main Visual FoxPro window or to the active user-defined window.
