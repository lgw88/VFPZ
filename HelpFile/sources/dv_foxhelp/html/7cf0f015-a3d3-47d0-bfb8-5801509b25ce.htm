How to: View Open Tables in Data Sessions
See Also 

You can see the tables that are currently open in a data session. For more information about data sessions, see How to: Use Data Sessions.

To see open tables in a data session

  • On the Window menu, choose Data Session.

    -OR-

  • Use the SET command without any arguments.

The Data Session window opens and displays name of the current data session, the table aliases for all open tables, and other options.

For more information, see SET Command Overview and Data Session Window.
