How to: Build a Function Expression
See Also 

In the Expression Builder, you can build an expression that contains a function.

To build a function expression

  1. When available, click the ellipsis (...) button to open the Expression Builder.

  2. In the Functions area of the builder, select a function from one of the function lists.

  3. Click OK.

For more information, see Expression Builder Dialog Box.
