Working with Report Controls

Report controls and their settings provide a way for you to display data and information and customize its appearance in reports and labels.

Describes manipulating report controls, for example, how to select multiple controls as a group, resize them, and change their alignment and relative position to each other.

Describes setting format options for report controls.

Describes how to create a simple report quickly without opening the Report Designer.

Describes how to programmatically open the Report Designer to create a report.
