Visual FoxPro 9.0 SP2
Object is not an offline view (Error 2009)

Some commands and clauses require an offline view.

  • You tried to use the ONLINE or ADMIN clauses.

    You must provide an offline view with these clauses. For more information, see USE Command.

  • You tried to drop a view.

    Use DROPOFFLINE( ) only with offline views.
