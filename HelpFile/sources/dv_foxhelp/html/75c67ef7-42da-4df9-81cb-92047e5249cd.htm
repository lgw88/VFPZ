Visual FoxPro 9.0 SP2
BaseClass Property
See Also 
 

Contains the name of the Visual FoxPro base class on which the referenced object is based. Read-only at design time and run time.

Object.BaseClass

Return Value

cBaseClass

Contains the Visual FoxPro base class on which the object is based.

For more information about Visual FoxPro base classes, see Base Classes in Visual FoxPro.
