Visual FoxPro 9.0 SP2
Field "field name" is not associated with a base table (Error 2174)

This error is generated by the CursorAdapter Class when it attempts delayed memo fetching. The base table is specified with the CursorAdapter Tables Property. The memo field should be associated with the base table with the CursorAdapter UpdateNameList Property.
