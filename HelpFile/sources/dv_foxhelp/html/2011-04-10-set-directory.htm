Visual FoxPro 9.0 SP2
SET DIRECTORY Command
See Also

This command is undocumented in the original VFP9SP2 Help File.

Specifies the default drive and directory.

SET DIRECTORY TO [cPath]

Parameters

cPath

Specifies one of the following:

  • A drive designator

  • A drive designator with a directory name

  • A child directory name

  • Any of the above using Microsoft MS-DOS shorthand notation ( \ or ..)

Remarks

SET DIRECTORY changes the default directory to the directory you specify.

Tip

SET("DIRECTORY") returns the default drive and the directory.

SYS(5) returns the default drive.
SYS(2003) returns the default directory with no drive designator.
SYS(5) + SYS(2003) returns the default drive and the directory.
