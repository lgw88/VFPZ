Visual FoxPro 9.0 SP2
Overwrite Macro Dialog Box
See Also

Replaces an existing macro with a new one of the same name. This dialog box appears when you define an already existing key combination in the Record Macro Dialog Box and click OK.

Overwrite

Replaces the existing macro definition with the new definition.

Append Keystrokes

Adds keystrokes to the end of the existing macro. The insertion point is placed in the last entered text box so you can type the keystrokes to append.
