Menu Options Dialog Box
See Also

Allows you to specify code for a specific menu. You can access this dialog box by selecting Menu Options from the View Menu in the Menu Designer.

Name

Contains the name of the menu. By default, this is the same as the text in the Prompt column of the MenuDesigner. Change it by typing a new name in the Prompt column.

Procedure

Provides space for menu procedure code, or displays existing code. If code exceeds the displayed editing region, the scroll bar is activated.

Edit

Opens an editing window so you can create code there instead of in the MenuOptions dialog box. To activate the editing window, click OK in the MenuOptions dialog box.

For more information, see Designing Menus and Toolbars.
