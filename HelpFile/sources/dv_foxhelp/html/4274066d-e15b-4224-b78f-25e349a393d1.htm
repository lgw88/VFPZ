Visual FoxPro 9.0 SP2
Reports and Labels
See Also 
 

These commands make it possible for you to create reports and control output.

Use To

_ASCIICOLS System Variable

Specify the number of columns in an ASCII text file created with the ASCII keyword in the REPORT FORM command.

_ASCIIROWS System Variable

Specify the number of rows in an ASCII text file created with the ASCII keyword in the REPORT FORM command.

\ | \\ Command

Print or display lines of text. 

CREATE LABEL Command

Open the Label Designer to create a label in standard label format or to design a custom label. 

CREATE REPORT Command

Open the Label Designer to create a label in standard label format or to design a custom label. 

LABEL Command

Send labels from a label file to the specified output format.  

MODIFY LABEL Command

Open the Label Designer to create or modify a label. 

MODIFY REPORT Command

Open the Report Designer to create or modify a report.

REPORT FORM Command

Display or print a report specified by a report definition (.frx) file, for example, created with the CREATE REPORT or MODIFY REPORT command.
