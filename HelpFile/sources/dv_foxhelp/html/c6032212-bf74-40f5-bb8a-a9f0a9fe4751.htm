How to: Set Timeout Intervals for Connections
See Also 

You can set timeout values for the connection to the remote server if you need to adjust how Visual FoxPro interacts with the server.

To set timeout intervals

  1. Open the application project containing the view in the Project Manager.

  2. In the Project Manager, expand the Data node, the Databases node, and then the node for the database containing the remote data source connection you want.

  3. Click the connection name, and then Modify.

    The Connection Designer opens for the connection.

  4. In the Timeout intervals area, set the values you want.

  5. When you are finished, on the File menu, click Save to save the settings.

For more information, see Connection Designer.
