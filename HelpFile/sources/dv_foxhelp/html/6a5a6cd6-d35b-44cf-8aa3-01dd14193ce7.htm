Visual FoxPro 9.0 SP2
Query Destination Dialog Box
See Also

Lets you send the results of your query to seven different outputs.

This dialog box appears when you select Query Destination from the Query menu and is available only for queries.

Output destinations

Displays buttons for you to choose the type of output for the results of your query. The following table lists and describes the output types.

Button Description

Browse Button, Query Destination Dialog Box

Displays the results of your query in a Browse window.

Cursor Button, Query Destination Dialog Box

Saves the results of your query in a cursor.

Table Button, Query Destination Dialog Box

Saves the results of your query as a table file.

Screen Button, Query Destination Dialog Box

Displays the results of your query in the active output window.
