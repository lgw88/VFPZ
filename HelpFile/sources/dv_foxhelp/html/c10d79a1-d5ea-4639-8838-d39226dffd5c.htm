This sample contains the following classes.

Class Library Description

_splash

_dialogs.vcx

Provides a splash screen for an application.

_login

_dialogs.vcx

Provides a Login box for an application.

_aboutbox

_dialogs.vcx

Provides an About box for an application.

_keywords

_dialogs.vcx

Provides access to the keyword table (.dbf).

_locateitem

_dialogs.vcx

Displays Locate Item dialog boxes for an application.
