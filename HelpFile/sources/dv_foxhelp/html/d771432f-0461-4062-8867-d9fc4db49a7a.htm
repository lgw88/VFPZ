HTML Item Dialog Box
See Also

The HTML Item dialog box is part of the Web Publishing wizard. This dialog box makes it possible for you to specify additional items and their attributes for the Web page you create.

To access the HTML Item dialog box

  1. From Tools menu, choose Wizards, and then click Web Publishing.

  2. Navigate through the wizard until you reach Step 3.

  3. Click Advanced.

  4. In the Advanced dialog box, click Add.
Type

This drop-down list provides a list of HTML tag items that you can add to your page.

Location

This drop-down list makes it possible for you to specify where on the Web page you want the selected HTML item to appear — in the header, before data, or after data.

Text Formatting

  • Text   Specify the exact text you want to display in the location you selected in the Location drop-down list.

  • Heading   Specify, from the drop-down list, the HTML heading tag you want to apply to the specified text.

  • Bold (Strong)   Specify whether you want the specified text to be formatted bold.

  • Italic   Specify whether you want the specified text to be formatted italic.

  • Font name   Specify the font you want to use for the specified text. If you chose a Heading, then this option is disabled, and your Web page will use the heading value set in the viewing browser.

  • Size   Specify the font size you want to use for the specified text. If you chose a Heading, then this option is disabled, and your Web page will use the heading value set in the viewing browser.

  • Alignment   Specify the left-to-right page alignment of the specified HTML item.

  • Forecolor   Specify the foreground (default text) color of the specified HTML item.

  • Backcolor   Specify the color of the specified HTML item.

Attributes

  • File/HREF   Specify or locate the file (or URL) you want to use if you chose Hyperlink or Image item in the Type drop-down list.

  • Style Class   Specify a path reference to a class contained in a cascading style sheet (.css).

When you click OK, you are returned to the Advanced dialog box where you can specify another HTML item, save the current data layout and style information, or proceed to Step 4 of the Web Publishing wizard.
