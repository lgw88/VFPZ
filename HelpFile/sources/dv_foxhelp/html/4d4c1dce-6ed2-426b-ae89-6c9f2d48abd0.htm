Visual FoxPro 9.0 SP2
Mail Merge Wizard
See Also

The Mail Merge wizard creates either a data source, such as names or addresses, for a Microsoft Word merged document, such as a form letter, mailing labels, envelopes, or catalogs, or a text file that can be used by any word processor.

Visual FoxPro Setup can install ODBC functionality through the Visual FoxPro OLE DB Provider. For information about merged documents, see Microsoft Word Help. For more information, see OLE DB Provider for Visual FoxPro.

To access the Mail Merge wizard

  • From the Tools menu, choose Wizards, and click Mail Merge.

Step 1 – Select Fields

This step makes it possible for you to specify the fields that contain the information you want to use with the merged document, which can be a Word document or a file from another word processor. You can select fields from a single table or view only.

Step 2 - Choose Word Processor

In this step, you specify whether you will create a file for use with Word or a generic, comma-delimited file for use with another word processor.

You must have a current version of Word installed on your computer if you select Microsoft Word. If you select Comma-delimited text file, the wizard will skip Steps 3 and 4 and go straight to the Finish screen.

Step 3 - Select Document Type

This step makes it possible for you to either create a new document, such as a mailing list, or to add to an existing document, such as a roster.

Step 4 - Choose Document Style

In this step, you choose a type of document to create.

Form Letter

Makes it possible for you to create or to use a letter template into which the data from the specified Visual FoxPro file will be merged.

Label

Makes it possible for you to create or to use a label template into which the data from the specified Visual FoxPro file will be merged.

Envelope

Makes it possible for you to create or to use an envelope template into which the data from the specified Visual FoxPro file will be merged.

Catalog

Makes it possible for you to create or to use a file that contains labels, such as Item or Color or Catalog number, into which the data from the specified Visual FoxPro file will be merged.

Step 5 - Finish

This step makes it possible for you to save the data as a text file or to continue into Word to specify how the Visual FoxPro data is merged into the existing or new merged document.

If you have chosen to create a Word document, the wizard will open the document in Word. If you have chosen to create a text file, the wizard will prompt you to save your comma-delimited file.
