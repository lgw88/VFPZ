Visual FoxPro 9.0 SP2
Text Box Builder
See Also

The Text Box builder makes it easy to set properties for a text box control. The text box is the basic control that allows users to add or edit data in a Character, Numeric, or Date field in a table. You set properties by selecting options displayed in the Text Box Builder dialog box.

To use the Text Box builder

  1. Using the Form Controls toolbar, place a text box control on a form.

  2. Select and right-click the text box control, and then choose Builder from the Form Designer shortcut menu.

  3. Choose the options you want from the dialog box, and then choose OK.

    When you choose OK, the builder closes, applying the property settings from every tab.

Builder Tabs

Format

Specifies miscellaneous format options for the text box and the type of input mask.

Style

Specifies text box visual effect, border, and character alignment.

Value

Specifies a field in which to store the text box's value.
