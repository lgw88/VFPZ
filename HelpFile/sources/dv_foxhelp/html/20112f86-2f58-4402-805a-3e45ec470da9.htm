XMLType Property
See Also 
 

Contains an object reference to the Microsoft XML Core Services (MSXML) 4.0 lSchemaType object, which contains restrictions, patterns, and so on for the field.

For more information about lSchemaType objects, see the MSXML 4.0 SDK on the MSDN Library Web site at http://msdn.microsoft.com/library/.

XMLField.XMLType

Return Value

Object reference. XMLType contains an object reference to the MSXML 4.0 lSchemaType object.

Remarks

Applies To: XMLField Class
