Reserved Words (Visual FoxPro)
See Also 
 

Reserved words in Microsoft Visual FoxPro include functions, system variables, properties, events, methods, commands, menu constants, and clauses. When programming, avoid using reserved words as names for window, table, or field names. If you use a reserved word as a name, it might generate a syntax error.

The following are all Visual FoxPro reserved words.
