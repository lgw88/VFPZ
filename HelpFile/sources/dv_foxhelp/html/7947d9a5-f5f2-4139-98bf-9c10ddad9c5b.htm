Unable to create temporary work files (Error 1410)

A SORT or INDEX command that requires temporary work files was not permitted by the operating system to create them.

  • Your directory is full.

    Remove some files.

  • Visual FoxPro does not have permission to access the temporary files directory.

    Verify your permissions.
