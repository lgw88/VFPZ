Visual FoxPro 9.0 SP2
_ASCIIROWS System Variable
See Also 
 

Specifies the number of rows in an ASCII text file created with the ASCII keyword in the REPORT FORM command. 

_ASCIIROWS = nExpression

Parameters

nExpression

Specifies the number of rows in the text file to create. The default value is 63.
