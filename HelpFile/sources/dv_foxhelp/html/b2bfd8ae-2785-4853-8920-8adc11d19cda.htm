Visual FoxPro 9.0 SP2
XMLTable XMLName property is an XPath expression (Error 2181)

  • The XMLAdapter Class ToXML Method generates this error if the XMLTable Class XMLNameIsXPath Property is set to true (.T.).

  • The XMLTable Class ApplyDiffgram Method and the ChangesToCursor Method generate this error if the XMLNameIsXPath Property is set to true (.T.).
