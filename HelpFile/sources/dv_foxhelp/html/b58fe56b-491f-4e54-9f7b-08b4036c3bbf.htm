Visual FoxPro 9.0 SP2
Quick Report Dialog Box
See Also

Makes it possible for you to select fields and a field layout for a report.

Field layout

Specifies whether to display fields from left to right across the page as columns or from top to bottom as rows.

Titles

Determines whether to display field names as titles in label controls that appear above or next to the corresponding field.

Add alias

Adds the alias to all field names automatically in the Report Designe.

Add table to data environment

Adds the table to the data environment automatically.

Fields

Displays the Field Picker dialog box so you can select the fields to include in the report. For more information, see Field Picker Dialog Box.
