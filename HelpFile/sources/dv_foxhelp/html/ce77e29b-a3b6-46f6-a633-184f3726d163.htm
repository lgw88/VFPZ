Visual FoxPro 9.0 SP2
Coverage Profiler Exit Dialog Box
See Also

Stops the Coverage Profiler application and enables you to choose how to continue or end coverage logging. This dialog box displays if you have issued SET COVERAGE Command FileName before running the Coverage Profiler application.

The dialog provides the following options:

Button Result

Yes

Enables you to continue logging data by appending more records to the current .log file for use in the Coverage Profiler application.

No

Enables you to continue logging data but overwrites the records in the current .log file.

Cancel

Leaves logging set off after closing the Coverage Profiler application.
