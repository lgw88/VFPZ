How to: Configure the Page Layout Grid for Reports
See Also 

You can use the page layout grid to help you position and align report and label objects. You can set the following options for the grid:

  • Display grid lines, which do not appear in the resulting output.

  • Activate snap-to-grid functionality, which makes it possible for you to align objects to grid lines automatically.

  • Change measurement units for the ruler.

  • Set grid spacing.

  • Show the position of objects in the grid.

To show or hide grid lines

  1. Open the report or label in the appropriate designer.

  2. On the View menu, click Grid Lines so that it is selected.

    The designer shows grid lines for positioning and aligning report and label objects.

To activate snap-to-grid

  1. Open the report or label in the appropriate designer.

  2. On the Format menu, click Snap to Grid so that it is selected.

    When you drag objects in the designer, they position themselves along grid lines.

    Note

    The visible grid lines provide guidance when you align controls in the layout. These lines do not determine the grid coordinates to which your controls snap when Snap to Grid is on. See Ruler/Grid Tab, Report Properties Dialog Box (Report Builder) for more information.

    Tip

    You can enable Snap to Grid for general guidance even for reports requiring more exact placement. To override the automatic alignment feature and position controls exactly, select them in the layout and use the arrow keys to move them. You can also use the default Report Builder's options to adjust Size and position in layout for individual report controls. See General Tab, Report Control Properties Dialog Box (Report Builder) for more information.

To change the measurement unit for the ruler

  1. Open the report or label in the appropriate designer.

  2. On the Format menu, click Set Grid Scale.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder or is set to a third-party builder, the Set Grid Scale dialog box displays or a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Set Grid Scale Dialog Box.

  3. In the Report Properties dialog box, click the Ruler/Grid tab.

  4. In the Ruler list, select the measurement unit you want.

  5. When you are finished, click OK.

For more information, see Ruler/Grid Tab, Report Properties Dialog Box (Report Builder).

To set grid spacing

  1. Open the report or label in the appropriate designer.

  2. On the Format menu, click Set Grid Scale.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder or is set to a third-party builder, the Set Grid Scale dialog box displays or a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Set Grid Scale Dialog Box.

  3. In the Report Properties dialog box, click the Ruler/Grid tab.

  4. In the Horizontal spacing and Vertical spacing boxes, type or select the spacing you want.

  5. When you are finished, click OK.

For more information, see Ruler/Grid Tab, Report Properties Dialog Box (Report Builder).

To show object position in the grid

  1. Open the report or label in the appropriate designer.

  2. On the View menu, click Show Position so that it is selected.

    When you move the mouse or drag an object across the grid, the position displays in the status bar.
