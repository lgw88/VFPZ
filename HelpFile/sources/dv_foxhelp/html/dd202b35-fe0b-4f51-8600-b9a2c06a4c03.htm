Visual FoxPro 9.0 SP2
"name" objects cannot return a value (Error 1983)

The root object of the form stores the return value from its Unload event with the TO clause of the DO FORM command.

  • Only Form and FormSet objects can return values.

    Make sure the root object is a form or a form set.
