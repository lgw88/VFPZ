How to: Add Shapes to Reports
See Also 

You can separate or emphasize parts of a report by adding shapes, such as lines, rectangles, and circles, to the page layout. For example, you can use lines to separate report details and information in the header and the footer. You can use rectangles as borders around report controls, report bands, or the entire page. You can add shapes to reports using the Line and Rectangle report controls.

To add a shape to the report page layout

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, point to Insert Control, and click Line, Rectangle, or Rounded Rectangle.

    The shape report control you chose appears in the Page Header band by default. To move the shape to another location, drag the shape to the location you want.

You can also add shape report controls at the position you want by selecting them on the Report Controls toolbar and drawing them. After you add the control, you can move it by dragging it to the position you want.
