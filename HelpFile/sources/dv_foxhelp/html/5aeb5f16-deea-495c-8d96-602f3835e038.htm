Visual FoxPro 9.0 SP2
Customize Toolbar Dialog Box (Visual FoxPro)
See Also

Allows you to add and delete toolbar buttons. The available buttons are displayed by category. Click a button to see its description.

This dialog box appears when you choose Customize in the Toolbars Dialog Box.

Categories

Lists the button categories.

Button

Displays the buttons included in the category you select in the Categories box. To add the button, drag it to the desired toolbar or drop it onto the Visual FoxPro desktop to create a new toolbar.

Description

Describes the selected button.

For more information, see How to: Configure Visual FoxPro Toolbars.
