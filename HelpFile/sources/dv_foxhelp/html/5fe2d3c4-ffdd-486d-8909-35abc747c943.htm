Visual FoxPro 9.0 SP2
How to: Customize the Toolbox
See Also 

The Toolbox can be customized to contain the categories and items that you use the most. You can also order items in a way that you find most effective. This can be done by adding or deleting a category, changing the order of categories, and adding or deleting items in a category.

Before you can customize the Toolbox, you must have it open.

To open the Toolbox

  • On the Tools menu, click Toolbox.

To add a new category

  1. Right-click the Toolbox, and click Customize Toolbox on the shortcut menu.

  2. In the Customize Toolbox dialog box, click the Categories folder.

  3. At the top of the Customize Toolbox dialog box, click Add Category.

  4. In the Add Category dialog box, type a new category name in the Category name box.

  5. In the Category type list, select the type of category you are adding, and click OK.

To delete a category

  1. Right-click the Toolbox, and select Customize Toolbox on the shortcut menu.

  2. In the Customize Toolbox dialog box, click the category you want to delete.

  3. At the top of the Customize Toolbox dialog box, click Remove Category.

  4. When prompted to delete the category, click OK.

To add an item to a category

  1. Right-click the Toolbox, and select Customize Toolbox on the shortcut menu.

  2. In the Customize Toolbox dialog box, select the category that you want to add your class.

  3. Click Add Item on the Customize Toolbox dialog box.

  4. In the Add Item dialog box, select the type of item you want to add, and click OK.

  5. In the Add Class Library dialog box, select the category to add to your Class Library.

  6. Click the ellipsis (...) button next to the Class Library box, select a class to add from the Open box, and click OK

  7. In the Add Class Library dialog box, click OK.

To delete an item from a category

  1. Right-click the Toolbox, and select Customize Toolbox on the shortcut menu.

  2. In the Customize Toolbox dialog box, select the category that you want to delete an item from.

  3. In the Category Item list, select the item that you want to delete.

  4. Click Remove Item on the Customize Toolbox dialog box.

  5. When prompted to delete the item, click OK.
