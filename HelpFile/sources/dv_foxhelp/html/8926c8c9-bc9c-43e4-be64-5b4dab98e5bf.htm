Visual FoxPro 9.0 SP2
Operating the Expression Builder

You can build expressions using the Expression Builder when access is provided in the Visual FoxPro user interface. In locations where you need to specify an expression in the Visual FoxPro user interface, an ellipsis (...) button is usually provided so you can open the Expression Builder.
