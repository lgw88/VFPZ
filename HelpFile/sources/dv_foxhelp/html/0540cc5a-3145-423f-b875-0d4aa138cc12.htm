Operators (Visual FoxPro)
See Also 
 

Operators make it possible for you to manipulate data of the same type. Visual FoxPro operators can be grouped according to the following data types and functions:

  • Character Operators

  • Date and Time Operators

  • Logical Operators

  • Relational Operators

  • Numeric Operators
