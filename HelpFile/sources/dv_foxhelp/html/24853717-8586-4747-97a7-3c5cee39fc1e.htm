Visual FoxPro 9.0 SP2
View Tab, Options Dialog Box
See Also

Contains options for controlling the display of Visual FoxPro features.

When you choose Set As Default, which appears on every tab in the dialog box, Visual FoxPro saves all the options on all tabs.

Show

Status bar

Specifies whether Visual FoxPro displays the status bar at the bottom of the main window. Corresponds to the SET STATUS BAR Command.

Clock

Specifies whether the system clock appears in the status bar. You must choose the StatusBar option for this option to have effect. Corresponds to the SET CLOCK Command.

Command results

Specifies that Visual FoxPro displays the results of certain commands including APPEND FROM, AVERAGE, CALCULATE, COUNT, INDEX, SUM, and TOTAL, in the main Visual FoxPro window, the system message window, the graphical status bar, or a user-defined window. Corresponds to the SET TALK Command.

System messages

Specifies that Visual FoxPro displays certain system messages in the status bar. This option corresponds to the SET NOTIFY Command.

Open last project on startup

Open the most recently used project when starting Visual FoxPro. This option is unavailable if any of the following conditions exist:

  • The Resource File setting on the File Locations tab in the Options dialog box is empty. For more information, see File Locations Tab, Options Dialog Box.

  • The SET RESOURCE command is set to OFF. For more information, see SET RESOURCE Command.

  • On a computer running Windows XP or 2000, the systems policy for not keeping a history of recently opened files has been set.

Most Recently Used list contains

Makes it possible for you to specify the number of files displayed in the most recently used (MRU) list. The default setting is 4.

This option is unavailable if any of the following conditions exist:

  • The Resource File setting on the File Locations tab in the Options dialog box is empty. For more information, see File Locations Tab, Options Dialog Box.

  • The SET RESOURCE command is set to OFF. For more information, see SET RESOURCE Command.

  • On a computer running Windows XP or 2000, the systems policy for not keeping a history of recently opened files has been set.

List display count

Specifies the maximum number of items to display initially in IntelliSense drop-down list boxes. The default value is 15. The minimum value is 5.
