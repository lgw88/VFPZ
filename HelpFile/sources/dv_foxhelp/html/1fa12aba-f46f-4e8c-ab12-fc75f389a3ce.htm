Visual FoxPro 9.0 SP2
Move Item Dialog Box
See Also

Makes it possible for you to change the location of a menu or menu item.

The Move Item dialog box appears when you select a row in the Menu Designer or Shortcut Designer and click Move Item.

Move Group

Name

Displays the name of the menu or menu item.

Type

Displays the type of the menu or menu item.

To Location Group

To Location box

Displays a list of valid locations to which you can move a menu item.
