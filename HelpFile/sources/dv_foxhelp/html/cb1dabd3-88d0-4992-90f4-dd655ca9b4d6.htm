Visual FoxPro 9.0 SP2
Go To Record Dialog Box
See Also

Specifies which record to move the record pointer to. Type the record number in the Record Number box, or use the up and down arrows as required until the desired number displays in the Record Number box. Choose OK to return to the Browse window with the record pointer on the specified record.

This dialog box appears when you choose the Record # command from the Go to Record submenu.
