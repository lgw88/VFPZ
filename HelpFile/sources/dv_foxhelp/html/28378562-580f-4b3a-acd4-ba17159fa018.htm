Visual FoxPro 9.0 SP2
MENU Command
See Also 
 

 

Included for backward compatibility. Use the Menu and Shortcut Designers instead.

This command is undocumented in the original VFP9 help file,
but the following documentation was found in the FoxPro for Windows 2.6 help, and may still be valid:

Creates a menu system.

MENU BAR array1, expN1
	MENU expN2, array2, expN3
	[, expN4]
READ MENU BAR TO memvar1 memvar2 
	[SAVE]

Remarks


MENU is included for backward compatibility. Use the FoxPro Menu Builder to create menus.

These commands create a menu system. A menu system consists of a menu bar across the top of the main FoxPro window, or the active user-defined window menu pads. Each menu pad has an associated menu popup containing a list of options.

Commands
--------

MENU BAR array1, expN1
 MENU BAR is used to create the menu bar and pads. Create a two-dimensional array of memory variables to contain the menu pad prompts and the messages that are displayed when the menu pads are selected. The elements in the first column of the array specified with array1 are the menu pad prompts, and the elements in the second column of the array are the messages. Specify where the messages are displayed with SET MESSAGE.

 The number of menu pads that appear in the menu bar is specified with expN1.

MENU expN2, array2, expN3 [, expN4]
 Use a series of MENU commands to create the menu popups that are displayed when a menu pad is selected.

 The numeric expression expN2 specifies which menu pad in the menu bar the popup is assigned to. The one-dimensional array array2 contains the prompts for the options on the popups.

 If the first character of an array element is a backslash (\), the corresponding option can't be selected. If the element contains a backslash and a hyphen (\-), a separator is drawn between the popup option prompts. You can create a hot key for an option by placing a backslash and a less-than sign (\<) before the desired character of the option prompt. A hot key is a highlighted letter in FoxPro for MS-DOS or an underlined letter in FoxPro for Windows that you can type to immediately choose the desired option. In FoxPro for Macintosh, the KEYCOMP setting determines if a hot key is underlined.

 The number of options on the popup is specified with expN3.

 The number of options displayed on the popup is specified with expN4. If there are more options on the popup than can be displayed at one time, you can scroll through the options in the popup.

READ MENU BAR TO memvar1, memvar2
 This command activates the menu bar. The values initially contained in the memory variables or array elements memvar1 and memvar2 specify which menu pad and popup option are selected when the menu bar is activated. When you choose an option from a popup, the values stored to memvar1 and memvar2 correspond to the popup and popup option you chose. If you press Esc to exit the menu, memvar1 and memvar2 contain 0.

SAVE
 Include SAVE to display an image of the menu after an option is selected.

Example

Copy Code
*** A program example of popup MENUS ***
SET STATUS OFF
SET ECHO OFF
SET TALK OFF
SET MESSAGE TO 24
*** Initialize the array for the menu bar ***
DIMENSION TOP(3,2)
TOP(1,1) = '  FILE  '
TOP(2,1) = '  EDIT  '       
TOP(3,1) = '  DATA  ' 
TOP(1,2) = 'FILE SELECTIONS'
TOP(2,2) = 'EDIT SELECTIONS'      
TOP(3,2) = 'DATA SELECTIONS' 
*** Dimension and initialize an array for the File menu ***
DIMENSION M_FILE(7)
M_FILE(1) = '\
