Add Dynamic Style Condition Dialog Box (Report Builder)
See Also

Enables you to name a new dynamic style condition and then open the Configure Dynamic Properties dialog box so that you can configure the new condition.

You can open this dialog box by clicking Add in the Dynamics tab of the Report Control Properties dialog box.

How to: Dynamically Format Report Controls

Enter condition name

Enter a string value (without quotes) to identify the condition. You cannot rename the condition after you create it. When you click OK the Configure Dynamic Properties Dialog Box (Report Builder) opens.
