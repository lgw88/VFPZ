How to: Find Tables or Views in Databases
See Also 

When your database is associated with many tables and views in the Database Designer, you can quickly locate a specific table or view. When locating an item, such as a table or view, associated with an open database, Visual FoxPro searches the current database first. If the item is not found, Visual FoxPro searches in the default search path.

To find a table or view in a database

  1. Open the database in the Database Designer.

  2. On the Database menu, click Find Object.

  3. In the Find Table or View dialog box, select the table or view you want to locate.

  4. Click Find.

    The table or view you selected appears active in the Database Designer.
