How to: Change Colors in Report Controls
See Also 

You can change the colors in Field, Label, Line, Rectangle, and Rounded Rectangle controls. Foreground and background colors are available depending on the report control.

Note

Background colors might not display if the control is set to transparent. For more information, see How to: Change Opacity of Report Controls.

To change colors in report controls

  1. Open the report or label in the appropriate designer.

  2. In the designer, click the control you want to change.

  3. On the Format menu, click Foreground Color or Background Color depending on its availability for the control.

  4. In the Color dialog box, select the color you want, and then click OK.

You can also change foreground and background colors by double-clicking the report control to open the report control properties dialog box. Click the Style tab, clear the Use default foreground color or Use default background color check boxes, and then click the ellipsis (…) buttons to select different colors. For more information, see Style Tab, Report Control Properties Dialog Box (Report Builder).

Note

If the _REPORTBUILDER system variable is not set to the default Report Builder or is set to a third-party builder, the report control dialog box displays or a different dialog box might display. For more information, see _REPORTBUILDER System Variable.
