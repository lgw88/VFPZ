Data Tab, Application Builder
See Also

On this tab of the Application builder, you can specify the data source and the format for forms and reports you include in your application. The grid displays the tables you select for your application.

Database Wizard

Makes it possible to create a new database for your application using the Database wizard. When the wizard closes, the tables and views in the new database are listed in the grid.

Table Wizard

Makes it possible to create a new table for your application using the Table wizard. When the wizard closes, the tables and views in the new database are listed in the grid.

Select

Makes it possible to select an existing database, table, or view to use in your application.

Clear

Removes all tables from the grid.

Generate

Generates new reports or forms from the selected table, for all checked forms or reports, in the specified form style or report style.

Form Style

Makes it possible to select a form style to use for one or more tables listed in the grid.

Report Style

Makes it possible to select a report style for one or more tables listed in the grid.

To use different styles for each form or report, select the forms or reports and the styles, and then generate the reports. Repeat this procedure for as many different styles as you want in your application.

If you choose OK, the builder closes, applying the property settings from all tabs.
