Visual FoxPro 9.0 SP2
Print File Dialog Box
See Also

Makes it possible for you to browse for and select a file to print.

This dialog box appears when you click the ellipsis (...) button to the right of the File box in the Print Options dialog box. The file types are filtered by the file type you select in the Print Options dialog box. For more information, see Print Options Dialog Box.

Code Page

Displays the Code Page dialog box so you can attach the correct code page. If the file you want to print uses the extended character set, click Code Page and make the appropriate selections.
