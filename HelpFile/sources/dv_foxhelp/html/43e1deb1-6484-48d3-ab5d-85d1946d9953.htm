Visual FoxPro 9.0 SP2
Cursor Button, Query Destination Dialog Box
See Also

Saves the results of your query in a cursor with the name you assign it. The cursor is read-only, appears in the View window, and is available for browsing, reporting, or other uses until you close it. This command is available only for queries.

Cursor Name

Specifies the name of the cursor.
