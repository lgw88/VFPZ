Visual FoxPro 9.0
Language Reference for OLE DB Development
 

Visual FoxPro contains language you can use with the OLE DB Provider for Visual FoxPro.

In This Section

Visual FoxPro OLE DB Provider Data Support

Describes data types as well as file and code characteristics supported by the Visual FoxPro OLE DB Provider.

Supported Visual FoxPro Commands and Functions in OLE DB Provider

Lists Visual FoxPro language you can use against the Visual FoxPro OLE DB Provider and in stored procedures, triggers, rules, and default values.

Unsupported Visual FoxPro Commands and Functions in OLE DB Provider

Lists Visual FoxPro language not supported by the Visual FoxPro OLE DB Provider.

Supported OLE DB Objects

Describes OLE DB objects supported by the Visual FoxPro OLE DB Provider.

Standard and Custom OLE DB Properties

Lists OLE DB properties supported by the Visual FoxPro OLE DB Provider.
