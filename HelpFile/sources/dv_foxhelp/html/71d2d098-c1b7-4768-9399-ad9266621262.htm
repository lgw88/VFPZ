Visual FoxPro 9.0 SP2
Protection Tab, Report Properties Dialog Box (Report Builder)
See Also

Makes it possible for you restrict selected activities in the Report or Label Designer when modifying the layout in protected mode. For more information, see Setting Protection for Reports.

  • How to: Protect Report Layouts

In PROTECTED mode

Specifies the protection settings you can configure for the report or label.

Report can not be run or printed

Specifies that the user cannot print the report or label layout when modifying the report or label layout in protected mode.

Disables the Print… option on the File menu, and the Run Report options on the Report menu and the designer's right-click context menu.

Report can not be previewed

Specifies that the user cannot preview the report or label layout when modifying the report or label layout in protected mode.

Disables the Print Preview option on the File, Report, and designer's right-click context menus, and the Preview option on the View menu.

Data Environment can not be modified

Specifies that the user cannot modify the report layout's data environment when modifying the report or label in protected mode.

Disables the Data Environment… option on the View and designer's right-click context menus; Hides the Data Environment tab in the Report Builder's Report Properties dialog box; and disables the Load Data Environment… option on the Report menu.

Page Layout is not available

Specifies that the user cannot change the printer setup or page size, margins, etc, of the layout when modifying the report or label layout in protected mode.

Disables the Page Setup option on the File menu, and hides the Page Layout tab in the Report Builder's Report Properties dialog box.

Optional Bands can not be changed

Specifies that the user cannot change the number of detail bands, title or summary band, and other layout settings when modifying the report or label layout in protected mode.

Disables the Optional Bands option on the Report menu and hides the Optional Bands tab of the Report Builder's Report Properties dialog box.

Data Grouping can not be modified

Specifies that the user cannot change the data grouping settings of the layout when modifying the report or label layout in protected mode.

Disables the Data Grouping option on the Report menu and hides the Data Grouping tab of the Report Builder's Report Properties dialog box.

Report Variables can not be edited

Specifies that the user cannot view or modify report variables when modifying the report or label layout in protected mode.

Disables the Variables option on the Report menu and hides the Variables tab of the Report Builder's Report Properties dialog box.
