Visual FoxPro 9.0 SP2
Type Property (Visual FoxPro)
See Also 
 

Contain a character indicating the file type of a file in a project. Available at design time and run time.

Object.Type

Remarks

Applies To: File Object

The following table list the values the Type property can contain and the corresponding file types.

Value FoxPro.H Constant File Type and Extension

d

FILETYPE_DATABASE

Database, .dbc

D

FILETYPE_FREETABLE

Free table, .dbf

Q

FILETYPE_QUERY

Query, .qpr

K

FILETYPE_FORM

Form, .scx

R

FILETYPE_REPORT

Report, .frx

B

FILETYPE_LABEL

Label, .lbx

V

FILETYPE_CLASSLIB

Visual class Library, .vcx

P

FILETYPE_PROGRAM

Program, .prg

L

FILETYPE_APILIB

Visual FoxPro dynamic link library, .fll

Z

FILETYPE_APPLICATION

Application, .app

M

FILETYPE_MENU

Menu, .mnx

T

FILETYPE_TEXT

Text file, varies

x

FILETYPE_OTHER

Other, varies

These values are stored in the Type field of the project's .pjx file.
