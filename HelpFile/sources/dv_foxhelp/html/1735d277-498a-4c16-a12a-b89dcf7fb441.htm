Converts files created in FoxPro 2.6 and earlier versions of Visual FoxPro to the current Visual FoxPro format. This is necessary when you want to use these files with later versions of Visual FoxPro.

For information about converting these older files, see How to: Convert Earlier Visual FoxPro Files.

Retain Visual FoxPro 3.0 default property values

Applies only when converting Visual FoxPro 3.0 projects. Visual FoxPro format default property settings for fonts and colors are compliant with Windows 98. Sometimes they are different from current Visual FoxPro settings. If you select this option, your converted forms will retain the Visual FoxPro 3.0 property settings.

File

Visible when using the Converter.app application as the converter tool. You can specify the file you want to convert. To browse for a file, click the ellipsis (...) button.

Directory

Visible when using the Converter.app application as the converter tool. You can convert all the files in a directory. When you select this option, you can either specify or browse for the directory

SCX files

Enabled when you select Directory. Indicates whether you should convert the screen files in the directory.

VCX files

Enabled when you select Directory. Indicates whether you should convert the visual class library files in the directory.

Check subdirectories

Enabled when you select Directory. Indicates whether you should also convert the files in the subdirectories of the specified directory.

Continue

Completes the file conversion process. When you click Continue, the Visual FoxPro Transporter dialog box appears only when you convert Macintosh or MS-DOS files that have never before contained Windows records. For more information, see Visual FoxPro Transporter Dialog Box.
