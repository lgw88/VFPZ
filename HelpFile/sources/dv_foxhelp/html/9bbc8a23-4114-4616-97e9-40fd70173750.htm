Managing Report Controls

You can manage existing report controls by grouping them, moving them, and aligning them. You can manage each control inidividually, or group controls together to manage as a single unit. When you select a report control in the Report Designer or Label Designer, Visual FoxPro displays many report control management options that you can choose on the Format menu.
