Visual FoxPro 9.0 SP2
Report file "name" is invalid (Error 50)
See Also 

The report definition contains an error, such as:

  • The number of data groups or Detail bands in a report exceeds the specified limits.

  • The report file (frx) does not contain the required columns for an FRX table.

  • The report file (frx) does not include a record for a required band, such as a Page Header band.
