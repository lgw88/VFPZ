Visual FoxPro 9.0 SP2
Report Properties Dialog Box (Report Builder)
See Also

Makes it possible for you to set options for reports and labels.

This dialog box appears when you choose Properties from the context menu of a report layout window or when you click Properties on the Report menu.

Note

This dialog box is only displayed if the _REPORTBUILDER system variable is set to the default Report Builder application. If it is empty, or has been set to a third-party report builder, the native Visual FoxPro dialog box (or a third-party dialog box) may appear. For more information, see _REPORTBUILDER System Variable.

The following table lists the tabs that exist in the Report Properties dialog box.

Note

When modifying the report or label in protected mode, some or all tabs might not be available as specified by protection mode settings.

Tab Description

Page Layout Tab, Report Properties Dialog Box (Report Builder)

Makes it possible for you to specify page layout and printer setup settings for the report or label layout.

Optional Bands Tab, Report Properties Dialog Box (Report Builder)

Allows you to specify the band structure of the layout, such as whether it includes a Title, Summary, or additional Detail bands, and how they are processed when the layout is processed at run time.

Data Grouping Tab, Report Properties Dialog Box (Report Builder)

Makes it possible for you to create and specify how the data is grouped within the report or label layout.

Variables Tab, Report Properties Dialog Box (Report Builder)

Allows you to create variables that are initialized and assigned values as the layout is processed at run time.

Protection Tab, Report Properties Dialog Box (Report Builder)

Makes it possible for you to hide or show tabs in the Report Properties dialog box or prevent users from taking certain actions when modifying the report or label in protected mode.

Ruler/Grid Tab, Report Properties Dialog Box (Report Builder)

Makes it possible to specify layout-specific settings for the ruler and page layout grid in the Report Designer and Label Designer.

Data Environment Tab, Report Properties Dialog Box (Report Builder)

Allows you to load a data environment into the current layout, and specify data session settings.

Document Properties Tab, Report Properties Dialog Box (Report Builder)

Allows you to add, edit, or delete document properties for the report. Document properties are included only in HTML or XML reports and are rendered as meta tags in the output of a report.

Other Tab, Report Properties Dialog Box (Report Builder)

Makes it possible for you to add comments, user data, and code elements that you can use at report run time for reports in the Report Designer or Label Designer.
