Visual FoxPro 9.0 SP2
Find Table or View Dialog Box
See Also

Makes it possible for you to a table or view that you cannot locate easily in the Database Designer. This dialog box appears when you click Find Object on the Database menu. For more information, see How to: Find Tables or Views in Databases.

Tables or Views

Shows a list of objects in your database layout that you can find. If the table or view you want does not appear in this list, it has not been added to the database.

Find button

Selects and displays the object in the database schema.
