Visual FoxPro 9.0 SP2
Multiple Selection Dialog Box (Report Builder)
See Also

Allows you to set properties on multiple selected controls at once in the report layout. The Multiple Selection dialog box appears when you select more than one control in the report layout and double-click on one of the selected controls. (Alternatively, you can right-click on one of the controls to display the context menu and select Properties.)

See How to: Select Multiple Report Controls for more information.

Note

This dialog box is only displayed if the _REPORTBUILDER system variable is set to the default Report Builder application. If it is empty, or has been set to a third-party report builder, then the native Visual FoxPro property dialog box for the control that you double-clicked on may appear. For more information, see _REPORTBUILDER System Variable.

Tabs

The following table describes the tabs that appear in the Multiple Selection dialog box:

Tab Options available

Selection Tab, Multiple Selection Dialog Box (Report Builder)

Allows you to review the selected controls; sort by control type or location; and remove controls from the list if you decide they should no longer be included in the selection.

Properties Tab, Multiple Selection Dialog Box (Report Builder)

Allows you to set protection options or a conditional printing expression for the currently selected controls.
