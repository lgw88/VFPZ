You can compare how WHILE loops differ between Visual FoxPro and other programming languages.

Visual FoxPro BASIC

DO WHILE n < 100

n = n + n

ENDDO

Do While n < 100

n = n + n

Loop

Pascal C/C++

while n < 100 do

n := n + n;

while(n < 100)

n += n;
