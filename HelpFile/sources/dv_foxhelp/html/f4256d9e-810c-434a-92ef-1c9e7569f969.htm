Visual FoxPro 9.0 SP2
Customize Toolbox Dialog Box
See Also

Displays options to change settings for the Toolbox and to edit categories and items displayed in the Toolbox.

Selection Treeview

Displays the areas that you can modify and change settings for the Toolbox.

Options

Displays the general options for setting the Toolbox behavior.

Run associated builders

Specifies that builders are displayed automatically in the Forms Designer when you create a control that has a builder registered to it.

Display toolbox help text section

Specifies if the help text section in the Toolbox is displayed.

Show tooltips

Specifies whether ToolTips are displayed for category items.

Always on top

Specifies whether the Toolbox should appear in front of other windows.

Double-click to open an item

Specifies whether a hyperlink item in the Toolbox needs a double-click or a single-click to open it.

Set CurrentControl property when a control is dropped on a grid column

Specifies that Visual FoxPro should automatically change the value of the CurrentControl property when a control is dropped on a grid column from the Toolbox.

Prompt to remove Text1 control when dropping new control on a grid column

Specifies that Visual FoxPro should automatically remove the Text1 control from the grid column when a new control is dropped on a grid column from the Toolbox.

Allow toolbox to be minimized

Specifies if the Toolbox minimize control button should be displayed so that the Toolbox can be minimized.

Category scroll speed

Specifies the speed, in milliseconds, that a category will scroll when the mouse is over a scroll up or down widget in the Toolbox.

Toolbox table

Specifies the table for the Toolbox to save settings.

Font

Sets the font attributes used for the Toolbox.

Clean Up Toolbox

Removes deleted items from the Toobox table.

Restore to Default

Restores the Toolbox to its default state.

Class Items

Displays options for how classes are displayed and used from the Toolbox.

Default naming convention for classes added to Toolbox

Specifies the naming convention for how a class is displayed in the Toolbox.

Drag and drop text template

Specifies what text will be added when you drag and drop a class from the Toolbox.

CTRL+Drag and drop text template

Specifies the text to be added when you CTRL+Drag a class from the Toolbox.

Edit

Opens an editor, which has IntelliSense and color coding, to change the drag-and-drop or CTRL+Drag text.

Filters

Displays options for managing your filters for the Toolbox.

New Filter

Creates a new filter and selects it for editing.

Delete Filter

Prompts to delete the currently selected filter.

Apply Filter

Applies the currently selected filter to the Toolbox.

Clear Filter

Removes the effects of a filter from the Toolbox.

Filter Name

Specifies the name for the currently selected filter.

Update

Saves any changes made to the filter name that were made in the filter name field.

Defined Filters

Displays a list of filters that can be applied to the Toolbox.

Select toolbox categories that belong to this filter.

Displays a list of filter options that can be applied with a filter.

Categories

Displays options for managing the categories in the Toolbox.

Add Category

Displays a dialog box to add a new category to the Toolbox.

Remove Category

Prompts to delete the currently selected category.

Category Properties

Displays a dialog box that contains information on the properties for the currently selected category.

Refresh Category

Updates the Toolbox with any changes made to the currently selected category.

Refresh Toolbox

Updates the Toolbox with any changes made to all of the categories displayed in the Toolbox.

Select an item to view

Specifies the type of items that are shown in the item list for a specific category.

Show checked items only

Specifies that only items in the item list that are selected and show in the category are visible.

Add Item

Displays a dialog box used to add a new item to the currently selected category.

Remove Item

Prompts to remove an item from the currently selected category.

Item Properties

Displays a dialog box that contains information on the properties for the currently selected item.

Sort Alphabetically

Sorts the items in the item list to be displayed in alphabetical order.
