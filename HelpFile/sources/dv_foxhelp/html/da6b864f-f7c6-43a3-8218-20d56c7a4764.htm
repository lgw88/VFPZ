Visual FoxPro contains functions in the following categories as well as additional functions.

Character Functions

Lists functions that operate on character data.

Data Conversion Functions

Lists functions that convert data from one data type to another.

Date and Time Functions

Lists functions that generate and manipulate date and time data.

COM Functions

Lists the commands and functions that make it possible for you to access COM objects and servers.

Numeric Functions

Lists functions that operate on or return numeric data.

XML Functions

Describes functions that make it possible for data manipulations between Visual FoxPro and Extensible Markup Language (XML) data.

Supported Visual FoxPro Commands and Functions in OLE DB Provider

Lists commands and functions you can use in the OLE DB Provider.

Unsupported Visual FoxPro Commands and Functions in OLE DB Provider

Lists commands and functions not supported in the OLE DB Provider.

For information about backward compatible-only language reference, see Backward-Compatible Language Elements.
