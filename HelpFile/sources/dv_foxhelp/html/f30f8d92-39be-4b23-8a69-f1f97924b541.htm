Visual FoxPro 9.0 SP2
Warning: No information is available to check remote update conflicts (Error 1496)

The ODBC driver does not report the number of records affected by an UPDATE, DELETE, or INSERT statement. Optimistic locking conflicts on remote views using this connection will not be reported.

Be sure your update code handles this limitation.
