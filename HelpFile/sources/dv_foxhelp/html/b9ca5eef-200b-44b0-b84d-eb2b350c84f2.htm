Visual FoxPro 9.0 SP2
New Local View Dialog Box
See Also

This dialog box lets you create a view of local tables or other views in the database using either the Local View wizard or View designer. Local views can include native Visual FoxPro .dbf format tables, existing local or remote views, any tables in a .dbf format, and other tables stored on local servers.

The New Local View dialog box only displays the Local View Wizard option when you have selected Prompt for Wizards in the Projects tab of the Options dialog box.

View Wizard

Opens the Local View wizard, which creates a local view.

New View

Opens the View designer so you can create a local view.
