Visual FoxPro 9.0 SP2
New Dialog Box
See Also

Makes it possible for you to create a new file. The New dialog box appears when you select New from the File menu.

Shortcut

CTRL+N

File Type

The following table lists the file types and the features available for creating them, depending on whether you click New File or Wizards.

File Type Created With

Project

Project Manager

Database

Database Designer

Table

Table Designer, Table Wizard

Query

Query Designer, Cross-Tab Wizard, Query Wizard, Local View Wizard, Remote View Wizard, PivotTable Wizard

Connection*

Connection Designer

View*

View Designer, Local View Wizard

Remote View*

Remote View Wizard

Form

Form Designer, Form Wizard, One-to-Many Form Wizard

Report

Report Designer, Report Wizard, One-to-Many Report Wizard

Label

Label Designer, Label Wizard

Program

Editing Window

Class

Class Designer

Text File

Editing Window

Menu

Menu Designer

* You must have a database open to enable this option.

New File

Displays a dialog box, designer, or editing window so you can create the selected file type.

Wizard

Opens a wizard that creates the selected file type. If there is a group of wizards to choose from, Visual FoxPro will prompt you to select the wizard you want.
