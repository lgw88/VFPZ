Visual FoxPro 9.0 SP2
BEGIN TRANSACTION command failed. Nesting level is too deep (Error 1590)

Transactions may only be nested 5 levels deep.

  • You must end a transaction before another transaction can begin.

    Call END TRANSACTION or ROLLBACK first.
