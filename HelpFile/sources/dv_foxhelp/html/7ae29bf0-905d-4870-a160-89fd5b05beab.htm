How to: Close Tables in Work Areas
See Also 

You can close tables in work areas within data sessions.

Tip

You can also close the currently open table in a work area automatically by opening another table in the same work area.

To close a table in a work area

  1. On the Window menu, choose Data Session.

    The Data Session window opens.

  2. In the Alias list, click the alias for the table you want to open, and then Close.

To close a table in a work area programmatically

  • Use the USE command without arguments or with only the IN clause and the work area number of the table you want to close.
