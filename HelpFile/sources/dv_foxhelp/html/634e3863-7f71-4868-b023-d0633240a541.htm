Returns Microsoft Visual FoxPro system information or sets its behavior.

SYS( )

Return Value

Character, Numeric, or Object reference

Remarks

Visual FoxPro SYS( ) functions return character or numeric values or object references that contain useful system information.

These functions are also used to set various Visual FoxPro system settings.
