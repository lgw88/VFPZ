Visual FoxPro 9.0 SP2
HOME( ) Function
See Also  Example
 

Returns the names of the Visual FoxPro and Visual Studio directories.

HOME([nLocation])

Parameters

nLocation

Specifies that HOME( ) returns the name of a specific Visual FoxPro or Visual Studio directory. Omitting nLocation is identical to HOME(0).

The following table lists the values for nLocation and the directory that HOME( ) returns. The directories listed below assume that you chose the default installation directory when you installed Visual FoxPro or Visual Studio.

nLocation Directory

0 or omitted

The directory from which Visual FoxPro was started

1

The directory in which Visual FoxPro is installed

2

The directory containing the Visual FoxPro samples. This directory is identical to the samples directory in the _SAMPLES system variable.

3

Visual Studio Common directory

4

Visual FoxPro Graphics directory

5

Visual Studio MSDN directory

6

Visual Studio Tools directory

7

Visual FoxPro subdirectory in the user's Visual FoxPro Application Data directory

8

Visual FoxPro Projects subdirectory in the user's Visual FoxPro My Documents directory. This directory is the default location for new projects and files created by Visual FoxPro.

Return Value

Character data type. HOME( ) returns the names of specific Visual FoxPro and Visual Studio directories.

Example

 Copy Code
CLEAR
? 'Visual FoxPro startup directory:           ', HOME( )
? 'Visual FoxPro installation directory:      ', HOME(1)
? 'Visual FoxPro samples directory:           ', HOME(2)
? 'Visual Studio common directory:            ', HOME(3)
? 'Visual FoxPro graphics directory:          ', HOME(4)
? 'MSDN samples directory:                    ', HOME(5)
? 'Visual Studio tools directory:             ', HOME(6)
? 'Visual FoxPro user app data subdirectory:  ', HOME(7)
? 'Visual FoxPro user documents subdirectory: ', HOME(8)
