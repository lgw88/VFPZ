Supported Visual FoxPro Commands and Functions in OLE DB Provider
See Also 
 

Visual FoxPro contains language you can use with the OLE DB Provider for Visual FoxPro. Some of these language elements are for use only against the OLE DB Provider; while others can be used in stored procedures, triggers, rules, and default values.

Note

You can determine the state of many SET commands, supported and unsupported except for SET ANSI, by querying the database. For example, the following line of code returns the value for SET PATH in the resultant cursor:

Note

SELECT SET('PATH') FROM Table WHERE RECNO()=1

Commands Called Against the Visual FoxPro OLE DB Provider

The Visual FoxPro OLE DB Provider supports native Visual FoxPro language syntax for commands you can execute against the provider. The following tables list these commands.
