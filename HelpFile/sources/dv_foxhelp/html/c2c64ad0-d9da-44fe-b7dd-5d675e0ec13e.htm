Visual FoxPro 9.0 SP2
Project Information Dialog Box
See Also

Lets you view and edit information about the project and its files. This dialog box appears when you select Project Information from the Project menu.

Shortcut

CTRL+J in the Project Manager or ALT+ENTER.
