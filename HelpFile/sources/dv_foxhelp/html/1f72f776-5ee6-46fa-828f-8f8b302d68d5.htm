Visual FoxPro 9.0 SP2
New Label Dialog Box
See Also

Displays predefined Avery label types. This dialog box appears when you select Label from the New dialog box and then choose New File. The type you select becomes the basis for the new label you create in the Label Designer. The Columns column refers to how many columns of that label type will fit on a page.

The following columns display information about the label (you cannot change these settings):

Avery No.

The manufacturer's identification number for a particular label type.

Height

The height in inches of an individual label.

Width

The width in inches of an individual label.

Columns

How many columns of that label type will fit on an 8 1/2 inch x 11 inch page.

Note

You can add a new label size to the list using the ADDLABEL.APP application located in the Visual FoxPro ...\Tools directory.
