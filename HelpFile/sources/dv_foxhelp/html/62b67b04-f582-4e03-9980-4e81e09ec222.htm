How to: Run Visual FoxPro Solution Samples
See Also 

You can run Visual FoxPro Solution samples from the Visual FoxPro Task Pane Manager, which includes functionality for searching and installing additional samples.

To run Solution samples

  1. To open the Task Pane Manager, select Task Pane from the Tools menu.

  2. From the Task Pane Manager toolbar, select Solution Samples.

  3. From one of the sample categories, choose a sample that you want to run.

  4. To run the sample, click the hyperlink for the sample. To see code for the sample, click the Click here to view the code icon located to the right of the sample description.

For more information about the Task Pane Manager, see Task Pane Manager.

You can also run the Solution samples from the Solution sample application located in ...\Samples\Solution\Solution.app.
