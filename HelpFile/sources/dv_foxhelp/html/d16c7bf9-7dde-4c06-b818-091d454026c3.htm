Add Class to Project Dialog Box

Adds a class from the Component Gallery to a project.

Add class to project

Adds the class to the project.

Create new class from selected class

Creates a new class and then adds the new class to a project. The New Class dialog box is displayed, allowing you to specify a name for the new class and the class library in which the new class is stored.

Create new form from selected class

Adds a form to the project. This option is only available adding a form to the project.
