Unsupported Visual FoxPro Commands and Functions in OLE DB Provider
See Also 
 

Certain Visual FoxPro commands and functions are not supported by the OLE DB Provider. The OLE DB Provider supports data-related Visual FoxPro language, and does not support object or user interface language.

The following tables list the most commonly used commands and functions not supported by the OLE DB Provider.
