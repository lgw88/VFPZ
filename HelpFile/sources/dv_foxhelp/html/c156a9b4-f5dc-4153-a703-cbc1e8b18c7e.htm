Visual FoxPro 9.0 SP2
Coverage Profiler Tool Buttons
See Also

Contains the following Coverage Profiler Tool buttons.

  Button Description

Open Catalog Dialog Box

Displays the Open dialog box so you can open a coverage log.

Save As Dialog Box

Displays the Save As dialog box so you can save the Coverage Profiler application information as a table (.dbf).

Coverage Profiler Statistics Dialog Box

Displays the Coverage Profiler Statistics dialog box so you can view summaries of the Coverage Profiler details and review Coverage Profiler statistics relative to a specific Visual FoxPro project.

Coverage Profiler Add-Ins Dialog Box

Displays the Coverage Profiler Add-Ins dialog box so you can specify Add-Ins to use with the Coverage Profiler.

Coverage Profiler Window

Displays coverage results in the lower pane of the Coverage Profiler window or in the Zoom window.

Coverage Profiler Window

Displays profiler results in the lower pane of the Coverage Profiler window or in the Zoom window.

Coverage Profiler Window

Displays the Coverage Profiler window as a single dialog box of two panes.

Coverage Profiler Window

Displays the Coverage Profiler as two separate dialog boxes that you can arrange and resize.

 

Coverage Profiler Find dialog box

Displays the Coverage Profiler Find dialog so you can search for specific item in the coverage log.
