How to: Align Report Controls Within the Layout
See Also 

You can align report controls in different ways. The following list describes some of the ways you can align report controls:

  • Relative to each other.

  • Relative to the grid in the designer.

  • Relative to the center of the band.

To align controls with each other

  1. Open the report or label in the appropriate designer.

  2. In the designer, select the report controls you want to align.

  3. On the Format menu, point to Align, and click the alignment that you want.

    Note

    The Center Vertically and Center Horizontally options apply to aligning report controls relative to the band they are in. If multiple controls are selected in multiple bands, then the alignment applies to the overall layout.

When aligning report controls relative to each other, Visual FoxPro uses the report control whose edge is the closest to alignment you select as the anchor.

Tip

Aligning horizontally placed report controls with their right side or left side cause them to overlap each other. Likewise, aligning vertically placed report controls with their top or bottom edge cause them to overlap each other. Therefore, align report controls with an edge that is open for all selected controls and away from each other.

To align controls with the designer grid

  1. Open the report or label in the appropriate designer.

  2. On the Format menu, ensure that Snap to Grid. is selected.

    Any control movement you now make in the report layout will be constrained by the designer grid.

To allow future control movement to be unconstrained by the designer grid, clear the Snap to Grid setting on the Format menu.

To center controls within a band

  1. Open the report or label in the appropriate designer.

  2. In the designer, select the report controls you want to align.

  3. On the Format menu, point to Align, and click Center Vertically or Center Horizontally.

    The report controls move to the vertical or horizontal center of the band that contains them.
