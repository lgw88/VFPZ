Visual FoxPro 9.0 SP2
SET STATUS BAR Command
See Also 
 

Displays or removes the graphical status bar.

SET STATUS BAR ON | OFF

Parameters

ON

(Default) Issue SET STATUS BAR ON to display the graphical status bar.

OFF

Issue SET STATUS BAR OFF to remove the graphical status bar.
