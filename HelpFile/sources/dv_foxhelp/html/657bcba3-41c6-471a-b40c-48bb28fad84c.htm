Visual FoxPro 9.0 SP2
Add Table or View Dialog Box
See Also

Appears when you select Query or Local View in the New dialog box and then choose New File. In this dialog box, you select one or more existing tables or views on which to base your new query or local view.

Database

Displays the current database by default and lets you select another database.

Tables/Views in database

Displays the tables or views from the current database, depending on whether you select the Tables or Views option.

Alias

Specify an alias instead of the automatically generated alias to use for the table or view you select.

Add

Adds the table or view.

Other

Displays the Open dialog box, from which you can select a free table to add to the query.

Tables

Displays the tables from the current database in the Tables in database box.

Views

Displays the views from the current database in the Views in database box.
