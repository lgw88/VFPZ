Visual FoxPro 9.0 SP2
Protection Tab, Report Band Properties Dialog Box (Report Builder)
See Also

Makes it possible for you to restrict certain actions for a report band in the Report Designer or Label Designer, preventing the user from changing the band when modifying the report or label in protected mode. For more information, see Setting Protection for Reports.

  • How to: Protect Report Bands

When in PROTECTED mode

Specifies protection options for the report band.

Band cannot be resized

Specifies that users cannot resize the report band when modifying the report or label in protected mode.

The Band cannot be resized check box is currently unavailable.

Properties dialog box is not available

Specifies that users cannot change properties of the band when modifying the report or label in protected mode.
