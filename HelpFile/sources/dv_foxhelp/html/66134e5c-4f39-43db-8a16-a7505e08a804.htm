Visual FoxPro 9.0 SP2
Page Setup Dialog Box (Visual FoxPro)
See Also

Specifies paper size and page orientation settings in Visual FoxPro. The settings available in this dialog box depend on the installed printer.

This dialog box appears when you select Page Setup from the File menu or from the Report Page Setup dialog box. For more information, see Report Page Setup Dialog Box.

Paper

Specifies paper settings for the page.

Size

Specifies the paper size.

Source

Specifies the paper source location in the printer.

Orientation

Specifies page orientation settings.

Portrait

Prints the page in a vertical position with the shortest side at the top.

Landscape

Prints the page in a horizontal position with the longest side at the top.

Margins

Margin settings in this dialog box are unavailable in Visual FoxPro.

Printer

Displays the printer Page Setup dialog box so you can specify printer settings.

Printer Page Setup Dialog Box

Specifies printer settings in Visual FoxPro. The settings available in this dialog box depend on the installed printer.

This dialog box appears when you click Printer in the Page Setup dialog box.

Printer

Specifies and displays printer information.

Name

Displays the names of the printers currently installed on your computer.

Status

Displays the printer status.

Type

Displays the model of the selected printer. The printer model is specified when you install the printer.

Where

Displays the network printer path.

Comment

Displays additional printer information, for example, printer location.
