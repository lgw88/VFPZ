How to: Set the Current Database
See Also 

You can a set a different database as the current database when more than one database is open. However, you can set only one open database as the current database.

To set the current database

  • Perform one of the following:

    • In the database list on the standard toolbar, select a database.

      -OR-

    • Use the SET DATABASE command.

The database you select is set as the current database.

For more information, see SET DATABASE Command.
