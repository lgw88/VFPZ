Run-time Extensions Dialog Box (Report Builder)

Makes it possible for you to assign code elements and expressions to a report control that you can leverage at report run time.

  • How to: Assign Structured Metadata to Report Controls

Run-time extensions

Specifies arbitrary code that ReportListener objects can execute.

Execute when

Specifies an expression that ReportListener objects can evaluate to determine whether to run the extensions. Clicking the ellipsis (…) button opens the Expression Builder so you can build an expression. For more information, see Expression Builder Dialog Box.
