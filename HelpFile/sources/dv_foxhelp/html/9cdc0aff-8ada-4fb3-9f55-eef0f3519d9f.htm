Visual FoxPro 9.0 SP2
Report Band Properties Dialog Box (Report Builder)
See Also

Makes it possible for you to set properties and options for a Band in a report or label page layout in the Report Designer or Label Designer.

This dialog box appears when you choose Properties from the context menu of a report band separator, when you double-click on a report band separator, or when you double-click on the band name in the Edit Bands Dialog Box.

Note

This dialog box is only displayed if the _REPORTBUILDER system variable is set to the default Report Builder application. If it is empty, or has been set to a third-party report builder, the native Visual FoxPro dialog box (or a third-party dialog box) may appear. For more information, see _REPORTBUILDER System Variable.

The following table lists each of the tabs that exist in the Report Band Properties dialog box. Some tabs may only be present for specific types of bands.

Note

When modifying the report or label in protected mode, some or all tabs might not be available as specified by protection mode settings.

Tab Description

General Tab, Report Band Properties Dialog Box (Report Builder)

Makes it possible for you to change the report band height, control whether the report band expands or contracts to fit data or adjust for suppressed blank lines, and expressions to evaluate before or after processing the band.

Band Tab, Report Band Properties Dialog Box (Report Builder)

(Detail bands only)

Allows you to specify how a Detail band is processed at run time.

Data Grouping Tab, Report Properties Dialog Box (Report Builder)

(Group Header and Group Footer bands only.)

Makes it possible for you to create and specify how the data is grouped within the report or label layout.

Protection Tab, Report Band Properties Dialog Box (Report Builder)

Makes it possible for you to set protection options for a report band in the Report Designer or Label Designer that restrict the user from making changes when modifying the layout in protected mode.

Other Tab, Report Band Properties Dialog Box (Report Builder)

Allows you to add comments and user data to a report band in the Report Designer or Label Designer.
