Visual FoxPro 9.0 SP2
Optional Bands Dialog Box
See Also

Makes it possible for you to include a Title, Summary, or additional Detail bands in a report or label.

Note

Depending on the setting of the _REPORTBUILDER system variable, this dialog may be replaced by an alternative user interface. For more information, see _REPORTBUILDER System Variable.

  • How to: Add Bands to Reports

  • How to: Configure Output for Report Bands

  • How to: Delete Bands from Reports

Report Title

Makes it possible for you to include a Title band and specify page break options for the Title band.

Title band

Specifies whether the layout includes a Title band for displaying information once at the beginning of the report.

New page

Specifies that a page break will be sent to the printer after the Title band information has rendered. Available only when the Title band check box is selected.

Report Summary

Makes it possible for you to include a Summary band and specify options for the Summary band.

Summary band

Specifies whether the layout includes a Summary band for displaying information once at the end of the report.

New page

Specifies that a page break will be sent to the printer before the Summary band information is rendered. Available only when the Summary band check box is selected.

Page header

Specifies that the Page Header band information will be rendered on the separate page along with the Summary band information.

Page footer

Specifies that the Page Footer band information will be rendered on the separate page along with the Summary band information.

Details

Makes it possible for you to add Detail bands.

Details

Specifies the number of Detail bands to include in the report. The default is set to 1. A report layout must contain at least 1 detail band.

Note

You can include a maximum of 20 Detail bands in a report.
