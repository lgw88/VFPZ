Visual FoxPro 9.0 SP2
Hierarchies Tab, Code Analyzer
See Also

This tab provides display of either the calling tree hierarchy or the class diagram of your program. The tab contains the following controls:

Tab options

Files

Displays either the calling tree hierarchy or the class diagram of your program.

Option

Selects the alternate displays of the Files outline control.
