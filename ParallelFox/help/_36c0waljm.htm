 parallel::CallMethod
Execute/call class method on worker. 
Parallel.CallMethod(cMethod, cClassName, cModule, [cInApplication], [lAllWorkers], [Parameter1], [Parameter2...])

Parameters

cMethod
Name of method to call on worker.

cClassName
Name of class containing method specified with cMethod.

cModule
Name of .VCX or .PRG containing class specified with cClassName.

cInApplication (optional for .PRG)
Name of .EXE or .APP containing class library specified with cModule.
NOTE: This parameter is required if cModule is a .VCX compiled into the main APP/EXE. Otherwise, ParallelFox will not be able to find the class.

lAllWorkers (optional)
Pass .T. to call method on all workers.

Parameters (optional)
Pass up to 21 parameters to cMethod.

Remarks

ParallelFox will instantiate cClassName on the worker, call cMethod, then release the object.

The return value of cMethod is passed as the first parameter of the Complete event.
