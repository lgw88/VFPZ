 worker::UpdateProgress
Send progress update to main process. 
Worker.UpdateProgress(nProgress, cMessage, [lForceUpdate])

Parameters

nProgress
Progress number/percentage complete.

cMessage
Progress message.

lForceUpdate (optional)
By default, update will not be sent to main process until Worker.ProgressInterval seconds have elapsed since last update.
Pass .T. to force immediate update and disregard Worker.ProgressInterval.

Remarks

Pass .T. as lForceUpdate when progress is complete to make sure main process is notified.
