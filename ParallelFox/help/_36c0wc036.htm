 worker::IsWorker
Returns .T. if currently running in Worker process. 
Worker.IsWorker()

Remarks

Use for programs that can run in either the main process or a worker, and when the behavior needs to vary depending on where it is running. For example, if a user previews a report, it will run in the main process. If the user prints the report, you could run it in the background.

Example

Local Worker as Worker
Worker = NewObject("Worker", "ParallelFox.vcx")

If Worker.IsWorker()
   * Code specific to worker process
Else
   * Code specific to main process
EndIf
