 parallel::SetWorkerClass
Change worker class from default.
Parallel.SetWorkerClass(cCOMProgID, cClass, cLibrary)

Parameters

cCOMProgID
COM ProgID for worker class.
Default is "ParallelFox.WorkerMgr".

cClass
Worker class to instantiate in Debug Mode.

cLibrary
Class library containing cClass for Debug Mode.
Must be accessible in main process path.

Remarks

If for some reason you want to override or rename the WorkerMgr class that controls the worker processes, call this method to tell ParallelFox the name of your class. Must be called before Parallel.StartWorkers().

Example

Parallel.SetWorkerClass("MyParallelExe.MyWorkerMgr", "MyWorkerMgr", "MyParallel.vcx")
