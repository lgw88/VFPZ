 parallel::StopWorkers
Stop worker processes. 
Parallel.StopWorkers([lStopImmediately])

Parameters

lStopImmediately (optional)
By default, ParallelFox waits until all pending commands are complete before stopping workers. Pass .T. to clear the command queue and stop workers immediately.

Remarks

This method should be called during your shutdown process. Any pending commands will finish running before workers are stopped, so you may want to notify users that the workers are shutting down.

When lStopImmediately is .T., pending commands are cancelled and there is no wait. However, commands currently running on the worker(s) may still complete.

Example

On Shutdown Do MyShutdown

Procedure MyShutdown
Local Parallel as Parallel
Parallel = NewObject("Parallel", "ParallelFox.vcx")

Parallel.StopWorkers()

* Perform more shutdown tasks
...

* Wait until worker shutdown is complete 
Wait "Shutting Down Workers..." Window Nowait
Parallel.Wait()

* Exit app
Clear Events

EndProc
