 worker::ReturnCursor
Return cursor to main process.
Worker.ReturnCursor([cAlias])

Parameters

cAlias
Alias of cursor to return to main process.

Remarks

Cursor will be returned and open in main process, even if main process does not use Parallel.BindEvent() to bind to ReturnCursor event. However, the current work area is not changed in the main process.
