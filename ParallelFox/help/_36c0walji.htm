 parallel::ExecScript
Execute script on worker. 
Parallel.ExecScript(cScript, [lAllWorkers], [Parameter1], [Parameter2...]

Parameters

cScript
String containing script to execute on worker.

lAllWorkers (optional)
Pass .T. to execute script on all workers.

Parameters (optional)
Pass up to 24 parameters to cScript.
Script must accept any parameters with Parameters/LParameters statement.

Remarks

The return value of cScript is passed as the first parameter of the Complete event.
