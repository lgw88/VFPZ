 worker::ReturnError
Return error to main process. 
Worker.ReturnError(nError, cMethod, nLine, cMessage, cCode)

Parameters

nError
Error number.

cMethod
Method/program containing error.

nLine
Line number containing error.

cMessage
Error message.

cCode
Line of code that caused the error.

Remarks

You worker error handler must populate the parameter variables before calling Worker.ReturnError().

Example

* Error handling inside worker
On Error Do MyWorkerErrorHandler with ERROR(), SYS(16), LINENO(1), MESSAGE(), MESSAGE(1)

Procedure MyWorkerErrorHandler
Lparameters nError, cMethod, nLine, cMessage, cCode
Local Worker as Worker
Worker = NewObject("Worker", "ParallelFox.vcx")

* Handle selected errors in worker silently
...

* Notify main process of other errors
Worker.ReturnError(nError, cMethod, nLine, cMessage, cCode)

EndProc
