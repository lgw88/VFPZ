 parallel::BindEvent
Bind to worker event. 
Parallel.BindEvent(cEvent, oEventHandler, cDelegate, [nFlags])

Parameters

cEvent
Name of the worker event you want to bind. See Worker Events for a list of available events.

oEventHandler
Specifies the object, which must be a valid Visual FoxPro object, handling the event.

cDelegate
Specifies the method, or "delegate", that handles the event for oEventHandler. The delegate method must have the same parameters as the event specified in cEvent.

nFlags (optional)
Specifies an additive bit flag you can set for the event binding operation.
See VFP BindEvent() Documentation for more info.
