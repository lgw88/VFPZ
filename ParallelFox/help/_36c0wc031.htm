 worker::ReturnData
Return data to main process. 
Worker.ReturnData(Parameter1, [Parameter2...])

Parameters

Parameters (optional)
Return up to 26 parameters to main process.

Remarks

Due to limitations with VFP's BindEvent() function, variables cannot be returned by reference, which means arrays are not supported. However, objects and collections can be returned.
