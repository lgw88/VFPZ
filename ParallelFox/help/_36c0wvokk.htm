 Installation
Installation of ParallelFox is simple:

  1. Download ParallelFox from VFPx.
  2. Unzip the file to a directory of your choice, such as C:\VFPX\ParallelFox.
  3. Open VFP, change that directory and enter DO INSTALL in the Command Window.

The install program registers ParallelFox.exe on your machine. On Windows Vista/7, administrator elevation will be requested to perform the registration.

You are also given the option to install Intellisense scripts for ParallelFox. This is recommended as it will improve your development experience. Two scripts are installed so you can quickly instantiate the ParallelFox classes when needed:

Entering "parallel" in a program expands to:

Local Parallel as Parallel
Parallel = NewObject("Parallel", "ParallelFox.vcx")

Entering "worker" in a program expands to:

Local Worker as Worker
Worker = NewObject("Worker", "ParallelFox.vcx")

ParallelFox also makes use of Doug Hennig's Favorites for IntelliSense, which Doug also used in the MY project for Sedna. This greatly simplifies the ParallelFox interfaces and provides extra details while you are coding.
