 parallel::Do
Execute program on worker. 
Parallel.Do(cPRG, [cInProgram], [lAllWorkers], [Parameter1], [Parameter2...])

Parameters

cPRG
Name of program or procedure to execute on worker.

cInProgram (optional)
Executes a procedure in the specified program file.
Equivalent of DO MyProcedure IN MyProgramFile.prg

lAllWorkers (optional)
Pass .T. to execute program on all workers.

Parameters (optional)
Pass up to 23 parameters to cPRG.

Remarks

FoxPro's DO command does not return a value, so .T. is passed as the first parameter of the Complete event. If you need the return value, use Parallel.Call() instead.
