 worker::EndCriticalSection
End Critical Section of code. 
Worker.EndCriticalSection(cCriticalSectionName)

Parameters

cCriticalSectionName
Name of critical section to end. Use same name passed to Worker.StartCriticalSection().

Example

Worker.StartCriticalSection("CreateMyTable")
If !File("MyTable.dbf")
   Create Table MyTable ...
EndIf
Worker.EndCriticalSection("CreateMyTable")
