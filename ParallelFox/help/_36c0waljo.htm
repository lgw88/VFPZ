 parallel::OnError
Set up global handler for worker errors. 
Parallel.OnError(cOnErrorCommand)

Parameters

cOnErrorCommand
Global error handler command.

Remarks

Available variables to use in cOnErrorCommand are nError, cMethod, nLine, cMessage, cCode.

Example

Parallel.OnError("Do MyErrorHandler with nError, cMethod, nLine, cMessage, cCode")
