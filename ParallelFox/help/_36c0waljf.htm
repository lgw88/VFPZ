 parallel::StartWorkers
Start worker processes. 
Parallel.StartWorkers(cProcedureFile, [cDirectory], [lDebugMode])

Parameters

cProcedureFile
The EXE, APP, or procedure file containing code to run on workers.

cDirectory (optional)
Directory to start worker in.
Defaults to current directory in main process.

lDebugMode (optional)
Pass .T. to start workers in separate, visible instances of Visual FoxPro for debugging.

Remarks

ParallelFox will only start workers in Debug Mode when the main process is running in the development version of VFP (_VFP.StartMode = 0).
